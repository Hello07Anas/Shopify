## SwiftCart iOS App

#### SwiftCart Client App
  - Brands
  - Categories
  - Orders
  - Home Design
  - Coupons (Discount Code)
  - Shopping Cart
  - Payment
  - Ads
  - Settings
  - Product Info
  - Search
  - Favorites
  - Authentication (Auth)

#### SwiftCartAdmin iOS App
  - Admin CRUD Products
  - CRUD Inventory
  - Admin CRUD Coupons
